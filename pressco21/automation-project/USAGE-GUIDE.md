## 1. 시스템 구조 한눈에 보기

```
[텔레그램 봇] ──→ [n8n 서버] ──→ [노션 DB]
                      ↑
              [구글 캘린더] (F2)
              [모닝 브리핑] (F3, 08:00)
              [밀린 업무 알림] (F4, 10:00)
```

| 워크플로우 | 기능 | 트리거 |
|-----------|------|--------|
| F1 | 텔레그램 → 노션 할 일 등록 | 메시지 수신 시 |
| F2 | 구글 캘린더 → 노션 할 일 등록 | 이벤트 생성 시 |
| F3 | 오늘 일정 모닝 브리핑 | 매일 08:00 |
| F4 | 기한 지난 업무 알림 | 매일 10:00 |

## 2. 노션 워크스페이스 구조

### 업무 카테고리 6개

노션의 모든 프로젝트와 할 일은 이 6개 카테고리로 분류됩니다.

| 카테고리 | 담당 영역 |
|---------|---------|
| 🎯 마케팅 | 유튜브·인스타·광고·콘텐츠 |
| 📦 상품/생산 | 상품 기획·DIY키트·원자재·레퍼런스 |
| 🛒 채널/판매 | 자사몰·스마트스토어·쿠팡·사방넷 |
| 💬 고객/CS | 쇼핑몰 CS·플랫폼 리뷰 |
| ⚙️ 운영/관리 | n8n 자동화·창고·재고 |
| 📊 경영/성장 | 매출 분석·트렌드·자기개발 |

### 프로젝트 20개 (PARA 분류)

> P = Project (기한 있는 프로젝트) / A = Area (지속 관리 영역) / R = Resource (참고 자료)

**🎯 마케팅**
- [A] 유튜브 채널 운영
- [A] 인스타그램 운영
- [A] 광고 운영
- [A] 상품 촬영/콘텐츠

**📦 상품/생산**
- [P] 시즌 상품 기획
- [P] DIY키트 라인업 확대
- [A] 원자재/재고 관리
- [R] 꽃/공예 레퍼런스

**🛒 채널/판매**
- [P] 웨딩기프트 런칭
- [P] 홈페이지 리뉴얼
- [P] 사방넷 세팅

**💬 고객/CS**
- [A] 쇼핑몰 CS 관리
- [A] 플랫폼 리뷰 관리

**⚙️ 운영/관리**
- [P] n8n 자동화 구축
- [A] 창고 정리

**📊 경영/성장**
- [A] 자기개발
- [A] 운동
- [A] 가계부
- [A] 월간 매출 분석
- [R] 경쟁사/트렌드 조사

### 노션 DB 계층 구조

```
업무 카테고리 DB
  └─ 프로젝트 DB  (카테고리에 연결)
       └─ 할 일 DB  (프로젝트 + 캘린더에 연결)
            └─ 캘린더 DB  (날짜별 묶음)
```

> **포인트**: 할 일에 프로젝트를 연결하면 → 업무 카테고리가 자동으로 롤업되어 캘린더 뷰에 정상 표시됩니다.

## 3. 텔레그램 봇 사용법

### 기본 형식

```
[프로젝트 - ]할 일 [@날짜 [시간]] [~목표날짜] [#메모]
```

> `[ ]` 안의 내용은 모두 선택사항입니다. 가장 간단하게는 할 일 이름만 입력해도 됩니다.

---

### 날짜 키워드

| 입력 | 의미 |
|------|------|
| `@오늘` | 오늘 |
| `@내일` | 내일 |
| `@모레` | 모레 |
| `@다음주` | 7일 후 |
| `@월` `@화` `@수` `@목` `@금` `@토` `@일` | 이번 주 해당 요일 (이미 지났으면 다음 주) |
| `@2/28` | 2월 28일 |
| *(날짜 생략)* | 오늘 |

---

### 시간 형식

| 입력 | 의미 |
|------|------|
| `14:00` | 오후 2시 |
| `9:30` | 오전 9시 30분 |
| `오전10시` | 오전 10시 |
| `오후3시30분` | 오후 3시 30분 |

---

### 커맨드 목록

| 커맨드 | 기능 |
|--------|------|
| `/목록` | 오늘 할 일 전체 보기 |
| `/완료 1` | 1번 항목 완료 처리 |
| `/상태 1 진행중` | 1번 항목 상태 변경 |
| `/밀린` | 기한 지난 업무 보기 |
| `/프로젝트` | 프로젝트 목록 전체 보기 |
| `/도움말` | 사용법 안내 |

**상태 목록**: 시작전 · 진행중 · 완료 · 거의완료 · 취소 · 핀 · 못함

## 4. 실전 사용 예시

### 빠른 할 일 등록

```
운동 @오늘
```
→ 오늘 날짜로 '운동' 할 일 등록 (프로젝트 미연결)

---

### 프로젝트 연결해서 등록

```
유튜브 - 섬네일 제작 @내일
```
→ '유튜브 채널 운영' 프로젝트에 '섬네일 제작' 등록

```
쇼핑몰 CS - 주문 문의 5건 처리 @오늘
```
→ '쇼핑몰 CS 관리' 프로젝트에 등록

> **팁**: 프로젝트 이름 전체를 타이핑하지 않아도 됩니다. 일부 키워드만 입력하면 자동 매칭됩니다.
> 예) `유튜브` → 유튜브 채널 운영 / `홈페이지` → 홈페이지 리뉴얼

---

### 시간 + 목표날짜 + 메모 모두 사용

```
광고 - 스마트스토어 입찰가 조정 @오늘 14:00 ~금 #클릭률 확인 후
```
- 오늘 오후 2시 시작
- 목표 기한: 이번 주 금요일
- 메모: '클릭률 확인 후'

---

### 개인 일정 등록

```
운동 @내일 오전7시
가계부 정리 @일
경쟁사 쿠팡 분석 @수 ~금 #신제품 런칭 시즌
```

---

### 프로젝트 목록 확인

```
/프로젝트
```
→ 카테고리별 20개 프로젝트 목록 표시

---

### 오늘 업무 확인 및 처리

```
/목록
```
→ 오늘 등록된 할 일 번호 목록 출력

```
/완료 2
```
→ 2번 항목 '완료' 처리

```
/상태 3 진행중
```
→ 3번 항목 상태를 '진행 중'으로 변경

---

### 밀린 업무 점검

```
/밀린
```
→ 기한이 지난 미완료 업무를 3일+ / 1~2일 경과로 구분해서 표시

## 5. 노션에서 확인하는 방법

### 세부 업무 캘린더 뷰

- **위치**: 노션 대시보드 → 세부 업무 캘린더
- **표시 기준**: 할 일의 `날짜(롤업)` 필드 (캘린더 DB 날짜 기반)
- **그룹핑**: 업무 카테고리별로 색상 구분

> 텔레그램으로 등록한 할 일이 캘린더에 표시되려면 **프로젝트 연결**이 되어 있어야 합니다.
> 프로젝트 없이 등록하면 할 일 DB에는 저장되지만 카테고리 그룹이 없어 탭에서 보이지 않습니다.

### 프로젝트별 진행 현황

- **위치**: 노션 대시보드 → 프로젝트 현황
- **확인 항목**: 진행률(수식), 상태 진행중/완료(롤업), 마감일

### 할 일 DB 직접 확인

- **위치**: 노션 → 할 일 DB
- **필터 활용**: 상태별, 프로젝트별, 날짜별 필터링 가능

## 6. 자동 알림 설정

### F3 모닝 브리핑 (매일 08:00)

텔레그램으로 오늘 할 일 목록을 자동 전송합니다.

```
📋 2/24(월) 오늘 할 일
━━━━━━━━━━━━━━━

⬜ 1. 섬네일 제작
⬜ 2. 주문 문의 처리
⬜ 3. 운동

📊 총 3건
```

### F4 밀린 업무 알림 (매일 10:00)

기한이 지난 미완료 업무가 있을 때만 알림을 보냅니다.

```
⚠️ 밀린 업무 알림
━━━━━━━━━━━━━━━━━━

🔴 3일+ 경과
  · 입찰가 조정 (5일)

🟡 1~2일 경과
  · 가계부 정리 (1일)
```

## 7. 자주 묻는 질문

### Q. 등록한 할 일이 노션 캘린더에 안 보여요

**원인**: 프로젝트를 연결하지 않으면 업무 카테고리 롤업이 비어서 캘린더 뷰 탭에 표시되지 않습니다.

**해결**: 등록 시 `프로젝트 - 할 일` 형식으로 프로젝트를 함께 입력하세요.

---

### Q. 프로젝트 이름이 정확히 기억나지 않아요

```
/프로젝트
```
를 입력하면 전체 목록을 볼 수 있습니다.

일부 키워드만 입력해도 매칭됩니다.
- `유튜브` → 유튜브 채널 운영
- `인스타` → 인스타그램 운영
- `CS` → 쇼핑몰 CS 관리

---

### Q. 날짜 없이 등록하면 어떻게 되나요?

날짜를 입력하지 않으면 **오늘 날짜**로 자동 등록됩니다.

---

### Q. 기존 할 일을 수정하거나 삭제하고 싶어요

현재 봇은 등록·상태변경만 지원합니다. 수정/삭제는 노션에서 직접 해주세요.

---

### Q. 새 프로젝트를 추가하고 싶어요

노션 → 프로젝트 DB에서 직접 추가하면 됩니다.
추가 후 텔레그램에서 해당 이름을 입력하면 자동 매칭됩니다.
