# 카카오 알림톡 템플릿 문안 (Phase 0 선작성)

> **작성일**: 2026-02-24
> **목적**: Phase 2 배포를 위한 알림톡 템플릿 사전 등록 (검수 3~5 영업일 소요)
> **등록 방법**: 메이크샵 관리자 > 회원관리 > 카카오 알림톡 > 템플릿 등록
> **발송 방식**: 메이크샵 오픈 API `send_sms` (건당 15.6원, SMS 0.7건 차감)

---

## ⚠️ 등록 우선순위

Phase 1b 시작 즉시 아래 순서로 등록 시작:

1. **F010** — 무통장 취소 알림 (Phase 1b 배포 전 필수)
2. **F024** — 리뷰 작성 유도 (Phase 2 초반 필수)
3. **F021** — 장바구니 이탈 유도 (Phase 2 핵심)
4. **F022** — 휴면 재활성화 (Phase 2)
5. **F020** — VIP 승급 알림 (Phase 2)
6. **F025** — 혜택 지급 알림 (Phase 2)

---

## 등록 시 주의사항

- 치환 변수는 반드시 `#{변수명}` 형식 사용
- 광고성 문구 최소화 (검수 반려 사유 1위)
- 실질적 정보 위주로 작성 (주문번호, 금액, 날짜 등)
- 문장 끝 "지금 바로" 같은 과도한 CTA는 피할 것
- 한 템플릿 최대 1,000자 이내

## F010 — 무통장 주문 자동 취소 알림

**template_code 후보**: `PRESSCO_CANCEL` (Phase 1b 배포 전 등록 필수)
**카테고리**: 주문
**발송 시점**: 무통장 입금 대기 72시간 초과 → n8n 자동 취소 처리 시

**치환 변수**:
- `#{고객명}`: 주문자 이름
- `#{상품명}`: 주문 상품명
- `#{주문번호}`: 메이크샵 주문 번호
- `#{취소일시}`: 자동 취소된 날짜/시간

---[템플릿 문안 시작]---
#{고객명}님, 주문하신 상품의 입금 기한이 만료되어 주문이 자동 취소되었습니다.

■ 주문번호: #{주문번호}
■ 상품명: #{상품명}
■ 취소 일시: #{취소일시}

입금 기한(72시간) 내에 입금이 확인되지 않아 취소 처리되었습니다.

재구매를 원하시면 동일 상품을 다시 주문해 주세요.

문의사항은 1:1 게시판이나 채널톡으로 연락해 주시면 빠르게 도와드리겠습니다.
---[템플릿 문안 끝]---

**검수 유의사항**: "자동 취소" 사실 명확히 기재. 재주문 유도는 CTA 버튼이 아닌 본문 안내로.

## F020 — VIP 고객 자동 승급 알림

**template_code 후보**: `PRESSCO_VIP`
**카테고리**: 회원
**발송 시점**: 월 1회 VIP 기준(3회 이상 구매 또는 15만원 이상) 충족 고객 자동 승급 시

**치환 변수**:
- `#{고객명}`: 회원 이름
- `#{등급명}`: VIP / VVIP 등 그룹명
- `#{쿠폰금액}`: 지급된 쿠폰 금액 (원)
- `#{유효기간}`: 쿠폰 사용 기한

---[템플릿 문안 시작]---
#{고객명}님, 감사합니다.

소중한 구매 이력이 인정되어 #{등급명} 회원으로 승급되었습니다.

■ 승급 혜택: 할인 쿠폰 #{쿠폰금액}원 자동 지급
■ 사용 기한: #{유효기간}까지

앞으로도 더 나은 서비스로 보답하겠습니다.

감사합니다.
---[템플릿 문안 끝]---

**검수 유의사항**: 쿠폰 금액은 치환 변수로. 고정 혜택 명시 시 실제와 불일치 위험.

## F021 — 장바구니 이탈 유도

**template_code 후보**: `PRESSCO_CART`
**카테고리**: 주문
**발송 시점**: 장바구니 담은 후 24시간 미구매 고객 (매일 10:00 n8n 스케줄)

**치환 변수**:
- `#{고객명}`: 회원 이름
- `#{상품명}`: 장바구니에 담긴 대표 상품명
- `#{상품수}`: 장바구니 상품 수량

---[템플릿 문안 시작]---
#{고객명}님, 장바구니에 담아두신 상품을 확인해 드립니다.

■ 담아두신 상품: #{상품명} 외 #{상품수}개

해당 상품은 현재 재고가 유지되고 있으나, 인기 상품은 품절될 수 있습니다.

필요하신 경우 지금 주문을 완료해 주세요.

문의사항은 1:1 게시판이나 채널톡으로 연락 주세요.
---[템플릿 문안 끝]---

**검수 유의사항**: "지금 바로" "빨리" 등 조급함 유발 문구 금지. 재고 현황은 실제와 다를 수 있으므로 "유지되고 있으나" 표현 사용.

## F022 — 휴면 고객 재활성화

**template_code 후보**: `PRESSCO_DORMANT`
**카테고리**: 회원
**발송 시점**: 마지막 구매일로부터 90일 경과 예정 D-7 (매일 스케줄)

**치환 변수**:
- `#{고객명}`: 회원 이름
- `#{마지막구매일}`: 마지막 구매 날짜
- `#{쿠폰코드}`: 복귀 할인 쿠폰 코드
- `#{쿠폰금액}`: 쿠폰 할인 금액 (원)
- `#{유효기간}`: 쿠폰 사용 기한

---[템플릿 문안 시작]---
#{고객명}님, 안녕하세요.

마지막 구매(#{마지막구매일}) 이후 한동안 방문이 없으셨네요. 잘 지내고 계신지요.

새로운 상품들이 많이 업데이트되었으니 둘러보시겠어요?

■ 특별 할인 쿠폰 #{쿠폰금액}원을 드립니다.
■ 쿠폰 코드: #{쿠폰코드}
■ 사용 기한: #{유효기간}까지

감사합니다.
---[템플릿 문안 끝]---

**검수 유의사항**: "휴면" "오랫동안" 등 부정적 표현 완화. 안부 인사 형식으로 자연스럽게.

**template_code 후보**: `PRESSCO_REVIEW`
**카테고리**: 주문
**발송 시점**: 배송완료 D+7 리뷰 미작성 고객 (n8n 스케줄)

**치환 변수**:

- `#{상품명}`: 구매 상품명

구매 후기를 남겨주시면 #{적립금액}원 적립금을 드립니다.

솔직한 사용 후기는 다른 고객분들께도 큰 도움이 됩니다.

적립금은 다음 구매 시 사용 가능합니다.

좋은 하루 되세요!
---[템플릿 문안 끝]---

**검수 유의사항**: 리뷰 작성 강요 표현 금지. "남겨주시면" 형태의 부드러운 권유 사용.

## F025 — 혜택 자동 지급 알림

**template_code 후보**: `PRESSCO_BENEFIT`
**카테고리**: 회원
**발송 시점**: 쿠폰/적립금/포인트 자동 지급 이벤트 발생 시

**치환 변수**:
- `#{고객명}`: 회원 이름
- `#{혜택종류}`: 쿠폰 / 적립금 / 포인트
- `#{혜택내용}`: 지급된 혜택 상세 (예: "5,000원 할인 쿠폰")
- `#{유효기간}`: 혜택 사용 기한

---[템플릿 문안 시작]---
#{고객명}님, 혜택이 지급되었습니다.

■ 혜택 종류: #{혜택종류}
■ 혜택 내용: #{혜택내용}
■ 사용 기한: #{유효기간}까지

마이페이지 > 쿠폰/적립금에서 확인하실 수 있습니다.

감사합니다.
---[템플릿 문안 끝]---

**검수 유의사항**: 지급 사유 명확히 안내하면 검수 통과율 높아짐. 필요 시 "#{지급사유}" 변수 추가.
