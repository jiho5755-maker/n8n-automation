# 쇼핑몰 운영 자동화 프로젝트

## 현재 진행 상황
- **Phase**: PRD 최종 완성 → 서버 배포 후 구현 시작
- **PRD**: `PRD.md` (전체 기능 명세 + 비용 모델 + 로드맵)
- **전략서**: `AUTOMATION-STRATEGY.md` (전체 전략 + 구현 로드맵)
- **의존성**: `automation-project/`의 Oracle Cloud + n8n 서버를 공유

## 프로젝트 요약
메이크샵 D4 쇼핑몰 운영 효율화를 위한 n8n 자동화.
CS 자동 기록/알림, 리뷰 AI 자동 답변, 미수금 관리, 고객 경험 자동화를
n8n + 메이크샵 자체 알림톡으로 구현. 챗봇은 Phase 3+ 선택 도입.

## 운영 환경
- **쇼핑몰**: 메이크샵 D4 (웨딩기프트/답례품)
- **판매 채널**: 자사몰 + 스마트스토어 + 쿠팡 + 11번가 등 (사방넷 연동)
- **CS 현황**: 채널톡 무료 플랜 (라이브챗) + 메이크샵 1:1 게시판 + FAQ 게시판
- **일 평균 CS**: 4~10건 | 주요 유형: 상품문의, 주문/결제
- **카카오 채널**: 이미 보유 (알림톡 연동 가능)

## 핵심 변경사항 (2026-02-20)
1. **SOLAPI 제거** → 메이크샵 자체 카카오 알림톡 (건당 15.6원, SMS 0.7건 차감)
2. **챗봇 Phase 1~2 미사용** → Phase 3+ 카카오 오픈빌더(추천) 또는 Typebot(선택)
3. **카카오 채널 이미 보유** → 개설/심사 불필요, 알림톡 템플릿 검수만 대기 (3~5영업일)
4. **서버 부담 감소** → Typebot 미설치로 1.5GB 절약. 4GB RAM + swap 4GB로 운영 가능
5. **SMS 단가 정정** → 공식 22원/건 (알림톡 15.6원 대비 29% 절감)
6. **장바구니 API 미확인** → 메이크샵 고객센터 문의 필수 (F021 실현 가능성에 영향)

## Phase별 핵심

### Phase 1a — CS 자동화 MVP (Week 1~3)
- 메이크샵 1:1 게시판 API → n8n 자동 수집 → 노션 CS DB 기록 + 텔레그램 알림
- CS 키워드 매칭 → 답변 템플릿 자동 추천 (노션 템플릿 DB)
- 미답변 CS 추적 (1시간/3시간/6시간 단계적 알림)
- 메이크샵 카카오 알림톡 설정 + 템플릿 등록

### Phase 1b — 리뷰 + 미수금 자동화 (Week 4~5)
- 자사몰 리뷰 AI 자동 답변 (메이크샵 API + OpenAI)
- 무통장 72시간 미입금 자동 취소 + 출고 알림톡

### Phase 2 — 고객 경험 고도화 (Week 6~8)
- VIP 자동 승급, 장바구니 이탈 유도, 휴면 재활성화, 혜택 자동 지급
- 주문 상태 알림톡 (메이크샵 관리자 설정, n8n 불필요)

### Phase 3+ — 마케팅 + 챗봇 선택 도입 (Week 9~)
- SNS 포스팅 리마인더, 리뷰→콘텐츠 제안
- [선택] 카카오 오픈빌더 FAQ 챗봇 (서버 부담 0, 추천)
- [선택] Typebot 자사몰 위젯 (Docker 1.5GB 추가, 필요 시)

## 기술 스택
- **n8n**: self-hosted Docker (핵심 엔진)
- **알림톡**: 메이크샵 자체 카카오 알림톡 (건당 15.6원)
- **AI**: OpenAI GPT-4o mini (리뷰 답변 + CS 템플릿)
- **채널톡**: 무료 플랜 라이브챗 유지
- **API**: 메이크샵 오픈, 네이버 커머스, 택배사 추적
