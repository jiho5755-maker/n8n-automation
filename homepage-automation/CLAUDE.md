# 쇼핑몰 운영 자동화 프로젝트

## 현재 진행 상황
- **Phase**: 전략 수립 완료 → 서버 배포 후 구현 시작
- **전략서**: `AUTOMATION-STRATEGY.md` (전체 전략 + 구현 로드맵)
- **의존성**: `automation-project/`의 Oracle Cloud + n8n 서버를 공유

## 프로젝트 요약
메이크샵 D4 쇼핑몰 운영 효율화를 위한 n8n 자동화.
CS AI 챗봇, 리뷰 AI 자동 답변, 경쟁사 모니터링, SNS 콘텐츠 관리를
n8n + Typebot으로 구현하여 비용 0~5,000원으로 운영.

## 운영 환경
- **쇼핑몰**: 메이크샵 D4 (웨딩기프트/답례품)
- **판매 채널**: 자사몰 + 스마트스토어 + 쿠팡 + 11번가 등 (사방넷 연동)
- **CS 현황**: 채널톡 무료 플랜으로 전 채널 통합 (라이브챗만, 챗봇 미사용)
- **CS 채널**: 채널톡(자사몰), 네이버 톡톡, 쿠팡윙, 인스타 DM, 카카오톡 채널, 전화/문자
- **일 평균 CS**: 4~10건 | 주요 유형: 상품문의, 주문/결제
- **채널톡 무료 제약**: 챗봇 시나리오 제한, 고객 정보 제한, 대화 기록 보관 제한

## 핵심 자동화 6개

| ID | 자동화 | 우선순위 | 핵심 |
|----|--------|---------|------|
| A1 | CS AI 챗봇 | P0 | Typebot 챗봇 → 채널톡 라이브챗 일원화 |
| A2 | CS 노션 DB + 미답변 추적 | P0 | 전 채널 CS 이력 자동 기록 + 시한 알림 |
| A3 | CS 템플릿 응답 | P1 | 키워드 매칭 → 답변 템플릿 자동 추천 |
| B1 | 리뷰 AI 자동 답변 | P1 | AI가 85~90% 자동 답변, 부정 리뷰만 직접 |
| C1 | 경쟁사 가격 모니터링 | P2 | 네이버/쿠팡 가격 변동 자동 감지 |
| D1 | SNS 포스팅 리마인더 | P2 | 노션 콘텐츠 캘린더 기반 알림 |

## 기술 스택 (추가분)
- **챗봇**: Typebot (Docker, self-hosted) - 자사몰 채팅 위젯
- **채널톡**: 라이브챗 유지 (버튼 숨김, Typebot에서 상담원 연결 시 자동 열림)
- **AI**: OpenAI API (리뷰 자동 답변, 월 ~5,000원 예상)
- **API**: 네이버 커머스, 쿠팡 Wing, 메이크샵 오픈, 사방넷

## Typebot + 채널톡 일원화 구조
```
자사몰 버튼 (1개, Typebot)
  └→ AI 챗봇 FAQ 자동 응대 (80~90%)
       └→ 상담원 필요 시 → 채널톡 라이브챗 자동 전환

다른 채널 (톡톡/쿠팡/카카오/인스타)
  └→ 채널톡 통합 (기존 그대로)

모든 CS → n8n → 텔레그램 알림 + 노션 CS DB 기록
```

## 노션 신규 DB (6개)
- CS 문의 DB: 전 채널 CS 이력 통합 기록
- CS 템플릿 DB: 답변 템플릿 관리 (초기 15개)
- 리뷰 DB: 3채널 리뷰 수집 + AI 답변 기록
- 콘텐츠 캘린더 DB: SNS 포스팅 일정 관리
- 경쟁사 상품 DB: 모니터링 대상 관리
- 가격 이력 DB: 일별 가격/순위 추이

## 구현 로드맵 (서버 배포 후)
- Week 2: CS AI 챗봇 + 채널톡 연동 + CS DB + 템플릿
- Week 3~4: 리뷰 AI 자동 답변 (스마트스토어→쿠팡→자사몰)
- Week 4: SNS 콘텐츠 캘린더 + 포스팅 리마인더
- Week 5~6: 경쟁사 가격 모니터링
- Week 7~8: 안정화 + 크로스 자동화 + 고도화
